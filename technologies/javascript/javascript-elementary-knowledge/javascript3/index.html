构造函数

系统自带构造函数

object就是一个系统自带的函数，是函数就可以执行object()，系统自带的构造函数可以批量的生产相似又独立的对象，new加上一个构造函数object的()执行， 就能够返回一个对象，接着用一个变量接受；这个方法和第一个方法建立的对象没有区别；

系统提供的其他构造函数有Array()，Number()等，像数字有原始值类型的数字，还有对象类型的数字；

自定义构造函数

构造函数和通常意义上的函数没有区别；自定义构造函数生成的对象和前两个没有区别；为了和一般的函数相区别，我们一般按照大驼峰式来写函数名，形如TheFirstName；

在用this定义方法的时候，里面不可以用自定义构造函数名，因为这个对象还没有建立；

构造函数内部原理

有new之后，会发生三个变化，分别是在函数体最前面隐式的加上this = {}；执行this.xxx = xxx；隐式的返回this。

一般不选择模拟自定义构造函数内部原理，因为有更深层的原理模拟不了；

可以用return来捣乱构造函数，但是一定要返回对象才行（像数组，函数和对象，不能是原始值）；

原型

原型的定义

原型是function对象的一个属性，它定义了构造函数制造出的对象的公有祖先。通过该构造函数产生的对象，可以继承该原型的属性和方法。函数一出生的时候其原型就被定义好了；原型本身也是对象；

原型的应用

利用原型特点和概念，可以提取共有属性；比如应用在代码冗余中；

原型的值可以修改和删除

构造器

英文全称，constructor；对象可以通过这个属性查看其构造函数是谁；

定义构造函数的父类

对象查看原型

通过隐式属性__proto__，其存储的是对象的原型；对象可以查看自己的原型是谁；

隐式命名规则前面和后面都有两条下划线，这是系统规定的；在一般开发中为了不让同事随意改动自己的变量，可以在变量前设置一个下划线；

在浏览器控制台输入person，返回一个对象，里面包含__proto__，这个__proto__里面还包含constructor和__proto__，值分别为Person()和Object

再次在浏览器控制台输入person，返回一个对象，里面包含__proto__；输入person.__proto__返回一个对象，里面包含constructor和__proto__以及一个name
