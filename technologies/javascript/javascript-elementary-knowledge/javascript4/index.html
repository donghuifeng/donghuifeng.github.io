什么是DOM

DOM，文档对象模型；

DOM定义了表示和修改文档所需的方法。

不能改变CSS指的是不能改变样式表，但是可以通过改变行间样式实现元素样式的改变；

对节点的增删改查

查看元素节点

document代表整个文档

html只是文档里的根标签；

document.getElementById()

id和元素是一一对应的，所以可以通过这个方法选择元素；

在IE 8以下的浏览器，不区分id大小写，而且也返回匹配name属性的元素

getElementsByTagName()

标签名，选出来的不是数组，而是类数组，既可以当数组使用，又可以当对象使用；从DOM开始，一切系统给生成的成组的东西，大部分都是类数组；这个方法在任何浏览器中都可以使用，兼容性很好；

getElementsByClassName()

这个不是所有浏览器都好使，比如IE8和IE8以下浏览器中根本没有，所以一般用getElementsByTagName()

getElementsByName()

name属性现在的浏览器都可以支持，但是在之前版本的浏览器中并不是在每个标签里面都好使；

querySelector()

css选择器，在IE7和IE7以下的版本中没有

querySelectorAll()

css选择器，在IE7和IE7以下的版本中没有

不是实时的

在所有浏览器都兼容，包含IE；

parentNode

父节点，最顶端的parentNode为#document

childNodes

子节点们

节点类型常见的有六种，分别是元素节点——1、属性节点——2、文本节点——3、注释节点——8、document节点——9、DocumentFragment——11；

firstChild

第一个子节点

lastChild

最后一个子节点

nextSibling

后一个兄弟节点

previousSibling

前一个兄弟节点

基于元素节点树的遍历

除了chilren之外，其他的IE9以下都是不兼容的；

parentElement

返回当前元素的父元素节点（IE不兼容）

children

只返回当前元素的元素子节点

node.childElementCount

当前元素节点的子元素节点个数（IE不兼容）

node.children.length

firstElementChild

返回的是第一个元素节点（IE不兼容）

lastElementChild

返回的是最后一个元素节点（IE不兼容）

nextElementSibiling/previousElementSibling

返回后一个/前一个兄弟元素节点（IE不兼容）

nodeName

元素的标签名，以大写形式表示，只读

nodeValue

Text节点或Comment节点的文本内容，可读写

nodeType

该节点的类型，只读

attributes

Element节点的属性集合
