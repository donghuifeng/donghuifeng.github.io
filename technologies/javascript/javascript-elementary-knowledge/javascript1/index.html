引导

javascript难在广度上，十分杂乱，并且难度是依次降低的

js的诞生

JavaScript最初的设计是为了解决Netscape Navigator浏览器的网页用户体验，并随浏览器在1996年诞生。

浏览器组成

由shell+内核两部分组成，能够看得到的都是shell；核心处理的部分是内核，其包含三大部分，分别是渲染引擎（语法规则和渲染）、js引擎和其他模块。浏览器的渲染是从上到下，16ms刷新一次。

js引擎

2001发布IE6，首次实现了对js引擎的优化和分离；

2008，Google发布最新浏览器引擎V8，可以将js代码直接转化为机器码执行，速度快；

js逼格

首先，js是解释性语言，解释一行执行一行，可以跨平台；

其次，js是单线程

完美兼容ECMA标准，也成为ECMAScript。

不足的地方是执行速度慢，但是随着浏览器越来越强大，速度方面得到了弥补。

js执行队列

计算机中不管程序是异步的还是同步，在底层都是一步一步执行的（轮转时间片）；其中计算机中的异步指的是同时执行，而同步指的是依次一个一个执行，这个概念和生活中的不一样。

开始学JS

JavaScript包含三大主要内容，分贝是ECMAScript, DOM和BOM。

如何引入js？

首先，是页面级引入，通过内嵌代码实现；其中内嵌的代码可以写到页面什么地方都是可以的；

其次，是外部文件引入，通过src属性来引入路径，type属性声明类型。

.htm是历史问题，文件后缀就三个字母

内部js可以引入外部js，但是这个时候，外部js有效，内部的无效。

js基本语法

变量

变量申明

一般情况下是将变量声明和变量赋值放在一起完成，其中的开发标准写法是用逗号分割，竖着写；

命名规则

首先，变量名必须以英文字母、_、$开头；

其次，变量名可以包括英文字母、_、$、数字；

最后，不可以用系统的关键字和保留字。

值类型

js值类型由值决定，其他语言由类型决定值。一个变量申明但是没有赋值，就是undifine；null表示占位的意思 ；

原始值

指的是不可改变的原始值，存储采用的是栈（stack）；常见的原始值类型有Number, String, Boolean, undefined, null；

引用值

指的是引用地址，存储采用的是堆（heap）；常见的引用值类型有Array, Object, Fuction, date, RegExp；

代码书写规范

错误分两种，分别是逻辑错误，其在扫描的时候没有发现，执行时候才能发现；另一种就是语法解析错误，属于低级错误，在扫描代码的时候就会发现

错误不会影响下面的错误。
